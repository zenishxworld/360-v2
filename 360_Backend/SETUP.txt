=============================================================================
  UniFlow Platform — EC2 Ubuntu Deployment Quickstart Guide
=============================================================================

This document details how to launch an Ubuntu EC2 instance, install all 
prerequisites, and run the UniFlow Platform deployment script.

-----------------------------------------------------------------------------
STEP 1: LAUNCH AN UBUNTU EC2 INSTANCE in AWS
-----------------------------------------------------------------------------
1. Log in to your AWS Console, navigate to the "EC2 Dashboard", and click
   "Launch instance".
2. Name: "uniflow-server"
3. OS: Select "Ubuntu" (Ubuntu Server 22.04 LTS, 64-bit).
4. Instance Type: Select "t3.medium" (minimum recommended) or "t3.large".
5. Key Pair: Create or select a key pair, download the ".pem" file, and save it.
6. Network Settings (Security Group):
   - Check "Allow SSH traffic".
   - Check "Allow HTTP traffic".
   - Click "Edit" and add two Custom TCP rules:
     * Port 8080 (Spring Boot Backend API) -> Source: Anywhere (or your IP)
     * Port 5678 (n8n Webhook UI)          -> Source: Anywhere (or your IP)
7. Click "Launch instance".

-----------------------------------------------------------------------------
STEP 2: SSH INTO THE EC2 INSTANCE
-----------------------------------------------------------------------------
Once the instance status is "Running", copy its Public IP and connect via terminal:

  chmod 400 path/to/key.pem
  ssh -i "path/to/key.pem" ubuntu@<YOUR_EC2_PUBLIC_IP>

-----------------------------------------------------------------------------
STEP 3: INSTALL PREREQUISITES ON EC2 (Ubuntu 22.04)
-----------------------------------------------------------------------------
Copy and paste this command block into your EC2 terminal to install Git, 
Java 17, Maven, Docker, and Docker-Compose:

  sudo apt-get update -y && \
  sudo apt-get install -y git openjdk-17-jdk maven curl ca-certificates gnupg lsb-release && \
  sudo mkdir -p /etc/apt/keyrings && \
  curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg && \
  echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null && \
  sudo apt-get update -y && \
  sudo apt-get install -y docker-ce docker-ce-cli containerd.io docker-compose-plugin && \
  sudo curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose && \
  sudo chmod +x /usr/local/bin/docker-compose && \
  sudo usermod -aG docker ubuntu && \
  newgrp docker

-----------------------------------------------------------------------------
STEP 4: CLONE REPO AND CREATE .env FILE
-----------------------------------------------------------------------------
1. Clone the project code:
     git clone <YOUR_GIT_REPO_URL>
     cd streamlined-suite

2. Create and edit your environment variables file:
     nano .env

3. Paste the following configuration template, filling in your own credentials:

     SPRING_PROFILES_ACTIVE=dev-aws
     PORT=8080

     SPRING_R2DBC_URL=r2dbc:postgresql://<db-host>:5432/<db-name>
     SPRING_R2DBC_USERNAME=<db-username>
     SPRING_R2DBC_PASSWORD=<db-password>

     SPRING_LIQUIBASE_URL=jdbc:postgresql://<db-host>:5432/<db-name>
     SPRING_LIQUIBASE_USER=<db-username>
     SPRING_LIQUIBASE_PASSWORD=<db-password>

     SPRING_DATASOURCE_URL=jdbc:postgresql://<db-host>:5432/<db-name>
     SPRING_DATASOURCE_USERNAME=<db-username>
     SPRING_DATASOURCE_PASSWORD=<db-password>

     JWT_SECRET=<secure_random_32_character_string>
     UNIFLOW_MASTER_VERIFY_TOKEN=<your_custom_verification_token>

     AWS_ACCESS_KEY_ID=<your_aws_access_key>
     AWS_SECRET_ACCESS_KEY=<your_aws_secret_key>
     AWS_S3_BUCKET_NAME=<your_s3_bucket_name>
     AWS_S3_REGION=<your_aws_region>

     RAZORPAY_KEY_ID=<your_razorpay_key_id>
     RAZORPAY_KEY_SECRET=<your_razorpay_key_secret>
     RAZORPAY_AMOUNT=100

     GOOGLE_OAUTH_CLIENT_ID=<your_google_client_id>
     GOOGLE_OAUTH_CLIENT_SECRET=<your_google_client_secret>

     UNIFLOW_AI_N8N_BASE_URL=http://localhost:5678
     UNIFLOW_AI_N8N_WEBHOOK_SECRET=uniflow-n8n-secret-2025
     N8N_WEBHOOK_URL=http://<YOUR_EC2_PUBLIC_IP>:5678
     ANTHROPIC_API_KEY=<your_anthropic_api_key>

   Press "CTRL+O" then "Enter" to save, and "CTRL+X" to exit the editor.

-----------------------------------------------------------------------------
STEP 5: RUN DEPLOYMENT COMMANDS
-----------------------------------------------------------------------------
Launch the backend databases/infrastructure and execute the deployment script:

  # Start Postgres, Redis, Kafka, and n8n
  docker-compose up -d postgres redis kafka n8n

  # Run compilation and start application
  chmod +x redeploy-via-git.sh
  ./redeploy-via-git.sh

-----------------------------------------------------------------------------
STEP 6: MONITOR LOGS
-----------------------------------------------------------------------------
Check output logs to verify successful boot:

  tail -f backend.log
